## インストール導線

現在はiPhoneでApp Storeアプリの検索画面を開くリンクにしています。
App Storeの個別アプリURLが発行されたら、`index.html` 内の `itms-apps://itunes.apple.com/jp/search?term=SongDice&media=software` を差し替えてください。
